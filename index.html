Конспект HTML CSS JS Для чайників

1. HTML

1.1 Теги - правопис

Теги пишуться таким способом: <тег><тег>
Також є одинарні теги, наприклад br або img. Вони пишуться <тег/>

Зверни увагу, що перед назвою в закриваючому тезі є знак "/"

1.2 Довідник тегів

    div - блок, в який можна класти інші елементи
    a - посилання
    img - картинка
    p - текст для параграфа
    h1 - заголовок(може бути на сайті тільки один)
    від h2 до h6 - заголовки поменше (їх може бути більше ніж 1)
    button - кнопка
    input - поле вводу тексту

    Тег input має атрибут type, і його значення text - звичайний, scale - лінія вибору яку треба тягнути, checkbox - галочка і т.д


    textarea - input, але більший
    link - посилання для підключення, наприклад, css файлів
    script - посилання для підключення, наприклад, js файлів, в самому тезі можна писати весь код, але це неефективно
    ul - список
    li - елемент списку, ставити в елемент списку

    1.3 Атрибути тегів

    У кожного тега є атрибути, вони записуються <тег параметр="значення">
    Кожен тег має атрибут class

    У CSS і JS класи позначаються з індексом ".", тобто, він ставиться перед назвою класу
    Щоб швидше створити елемент з класом, просто напиши тег.клас і натисни на підказку
